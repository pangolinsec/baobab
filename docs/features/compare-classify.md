# Compare & Classify

{: .warning }
This feature is **planned but not yet implemented**. No classification or comparison code exists in the codebase.

## Planned design

When implemented, this feature will enable scoring and comparing branches side-by-side, with optional ML-powered similarity analysis.

See [Feature 17 spec](https://github.com/OWNER/baobab/blob/main/Features/17-compare-classify.md) for the full planned design.
