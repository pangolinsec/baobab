# RAG Over Conversations

{: .warning }
This feature is **planned but not yet implemented**. No embedding, indexing, or retrieval code exists in the codebase.

## Planned design

When implemented, RAG will allow the model to retrieve relevant passages from past conversations and use them as additional context when generating responses.

See [Feature 19 spec](https://github.com/OWNER/baobab/blob/main/Features/19-rag.md) for the full planned design.
