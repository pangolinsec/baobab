# Search

Full-text search across all conversations and messages.

## Usage

Use the search bar at the top of the sidebar to search across all conversations, or use in-conversation search to find messages within the current tree.

## Capabilities

- **Case-insensitive** substring matching
- **Role filter**: search only user messages, only assistant messages, or both
- **Starred filter**: limit results to starred messages
- **Cross-conversation**: search across all conversations from the sidebar
- **Per-conversation**: search within the current conversation tree

## Search results

Results are displayed as a list of matching nodes with:

- A preview of the matching content
- The conversation title
- The node's role (user/assistant)

Clicking a result navigates to that conversation and selects the matching node.

## In-tree highlighting

When searching within a conversation, matching nodes get an **amber highlight ring** in the tree view, making them easy to spot in large trees.
