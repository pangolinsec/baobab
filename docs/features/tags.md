# Tags

Organize conversations with free-form tags.

## Usage

Add tags to any conversation to categorize and filter them. Tags are stored as a string array on the `Conversation` object.

## Features

- **Free-form**: type any tag name, no predefined categories
- **Autocomplete**: previously used tags are suggested as you type
- **Multiple tags**: conversations can have any number of tags
- **Tag pills**: tags are displayed as colored pills on conversation items in the sidebar
- **Filtering**: filter the sidebar conversation list by tag (coming in v2)

## Tag management

Tags are managed per-conversation through the conversation settings or sidebar. The tag input component supports:

- Typing a new tag and pressing Enter to add
- Clicking the X on a tag pill to remove
- Autocomplete suggestions from existing tags across all conversations
