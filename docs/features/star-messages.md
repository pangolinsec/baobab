# Star Messages

Bookmark important messages for quick access.

## Usage

Click the **star icon** on any node (in the detail panel or context menu) to toggle its starred status. Starred nodes:

- Display a star indicator on the tree node
- Appear in the **Starred** tab in the sidebar
- Are indexed in the database for fast lookup

## Starred sidebar view

The sidebar has a **Chats / Starred** toggle. In Starred mode, all starred messages are listed across all conversations, ordered by timestamp. Clicking a starred message navigates to that conversation and selects the node.

## Use cases

- Mark key insights or conclusions in a long conversation tree
- Create a reading list of important assistant responses
- Quickly jump back to decision points in complex explorations
